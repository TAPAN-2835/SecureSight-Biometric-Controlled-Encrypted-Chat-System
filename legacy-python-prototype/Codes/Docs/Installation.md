# Installation Guide

This project runs entirely on Python.  
No Ubuntu or VirtualBox required.

---

## 1. Install Python Libraries

Run in Command Prompt / Terminal:

pip install pycryptodome
pip install flask
pip install waitress

(If you want to convert to EXE)
pip install pyinstaller


---

## 2. Verify Crypto Library

python -c "from Crypto.Cipher import AES; print('Crypto OK')"
