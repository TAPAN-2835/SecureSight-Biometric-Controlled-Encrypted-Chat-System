# Installation Commands (Windows + Linux)

These are the exact commands needed to install dependencies for the AES/DES encrypted chat system.

---

## Install Required Python Libraries

pip install pycryptodome
pip install flask
pip install waitress

---

## Optional Tools (if you want EXE output)

pip install pyinstaller

---

## Verify Crypto Installation

python -c "from Crypto.Cipher import AES; print('Crypto Working')"

No other installations required.
